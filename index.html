"Germany reaching the final in 8 of 21 World Cups (38.1%) is the single most statistically robust pattern in football. I built the H2H prior matrix to reproduce this cadence organically — not by hardcoding an outcome, but by encoding the 18 rivalries that determine who Germany beats and who beats them. If the model doesn't produce ~35–42% finals rate, the priors are miscalibrated."

— Wahid Tawsif Ratul, WC 2026 Model Author

Football's Beautiful Contradictions

The Ironies

Six historically documented patterns embedded into the model's prior structure. Each one is ignored by conventional ELO — each one keeps happening.

Germany: The Finals Machine

8 finals in 21 WCs (38.1%)

Run simulation
H2H prior encodes Germany's KO dominance vs. Netherlands, England, France, Argentina, Brazil — the teams they historically dispose of.

Italy–Germany Curse

Italy 2W–0L in WC KO vs. Germany

Run simulation
4–3 AET (1970 SF) · 3–1 (1982 SF). Prior +0.24 log-scale for Italy — the highest single rivalry weight in the model.

Brazil's European Nightmare

5 of last 6 WC exits to European teams

Run simulation
France '06 · Netherlands '10 · Germany '14 (7–1) · Belgium '18 · Croatia '22. H2H: France +0.22, Germany +0.18 vs. Brazil.

England's Penalty Curse

~25–30% historical shootout win rate

Run simulation
Model uses pen_hist=0.40 — slightly above historical, reflecting Euro 2021 final and recent coaching improvements.

Host Nation Advantage

3 hosts in 2026 splits the effect

Run simulation
Historical host win rate: 23.1%. Since 1998: only France 1998. USA, Mexico, Canada split any 2026 home advantage three ways.

Defending Champions' Curse

No repeat winner since Brazil 1970

Run simulation
France '02 · Italy '10 · Spain '14 · Germany '18: all group stage exits. France '22 reached the final (exception). Argentina 2026?

"These six patterns are not edge cases — they are the recurring shape of World Cup history. FiveThirtyEight's SPI and Goldman Sachs's model treat each match as statistically independent of history. This model doesn't. The H2H matrix, penalty records, and form dynamics exist precisely to reproduce these ironies from first principles — not to hardcode them."

— Wahid Tawsif Ratul
